# Amazon Redshift Data Model Migration

## Problem
Enterprise data models needed to be migrated into Redshift while improving reporting performance and reliability.

## Approach
- Developed ETL workflows to migrate and optimize data models
- Implemented incremental data loading to support ongoing updates
- Validated business logic during migration to ensure accuracy
- Collaborated with stakeholders to align on reporting requirements

## Outcome
Improved reporting performance and data reliability through an optimized Redshift data model.

## Skills Demonstrated
Amazon Redshift · AWS Glue · SQL · Data Modeling · ETL · Stakeholder Collaboration

*Note: This reflects the structure and approach of production work; proprietary business logic and company data are not included.*
